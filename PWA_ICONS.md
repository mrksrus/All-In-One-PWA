# PWA Icons

The PWA requires icon images for installation. You need to create and add these files:

## Required Icons

1. `frontend/public/pwa-192x192.png` - 192x192 pixels
2. `frontend/public/pwa-512x512.png` - 512x512 pixels

## How to Create Icons

You can:
1. Use an online icon generator
2. Create them in an image editor
3. Use a tool like `pwa-asset-generator`

The icons should represent your app. Simple designs work best.
